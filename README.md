# Python Chat Application 💬

A real-time multi-client chat application built using Python sockets and Tkinter GUI.  
This project demonstrates networking, multithreading, and GUI development in Python.

The application allows multiple users to connect to a central server and communicate in real time with a modern messaging interface.

---

## 🚀 Features

- Real-time messaging using socket programming
- Multi-client support using multithreading
- Graphical User Interface built with Tkinter
- Chat bubbles similar to messaging apps
- User avatars (username initials)
- Emoji support
- File sharing notification
- Dark mode toggle
- Scrollable chat interface
- Username login system

---

## 🛠 Technologies Used

- Python
- Socket Programming
- Multithreading
- Tkinter (GUI Library)
- Networking Concepts

## ⚙️ How to Run the Project

### 1️⃣ Clone the repository
git clone https://github.com/rajsuryawanshi2005/python-chat-application.git

### 2️⃣ Navigate to the project folder
cd python-chat-application

### 3️⃣ Run the server
python server.py

### 4️⃣ Run the GUI client
python client_gui.py

Run multiple clients to simulate multiple users chatting.
